Actividad 2 CSS

Identifica y describe las propiedades de los siguientes elementos CSS:

Propiedad Valor Inicial Descripcion
color Black Color del texto
font-family Depende o Times New Roman lista de fuentes o familias de fuentes
font-size 16px Define el tamaño de la fuente
font-weight normal (400) Define el grosor de la fuente
font-style normal especifica los estilos de la fuente(normal, italic, oblique)
font-variant normal permite indicar si la fuente debe mostrar o no el estilo versalita.
font normal Es un atajo para establecer todos los valores de fuente
text-align left Alinea el texto
line-height 1.2 Espaciado entre lineas
text-decoration
  • color: currentcolor
  • style: solid
  • line: none
    		•  se usa para establecer el formato de texto a subrayado (underline) y suprarrayado (overline), tachado (line-through) o parpadeo (blink). El subrayado y el suprarrayado son posicionados bajo el texto, mientras la línea a través de éste se posiciona por encima.
    text-transform none especifica el cambio entre mayúsculas y minúsculas del texto de un elemento.
    vertical-align baseline especifica el alineado vertical de un elemento en línea o una celda de una tabla.
    text-indent 0 Establece la tabulación o indentación de la primera línea del texto de un elemento
    letter-spacing normal Espacio horizontal entre los caracteres
    word-spacing normal Espacio horizontal entre palabras/td>
    white-space normal determina cómo se maneja el espacio en blanco dentro de un elemento
